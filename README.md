# IMDb Movie Recommendation System Using Storylines

A Content-Based Movie Recommendation System that recommends movies based on **storyline similarity** using **Natural Language Processing (NLP)**, **TF-IDF Vectorization**, and **Cosine Similarity**.

The application scrapes IMDb 2024 movie data, preprocesses movie storylines, builds a recommendation engine, and provides a user-friendly Streamlit interface.

---

## Project Overview

This project recommends movies by analyzing the similarity between movie storylines.

Users can:

-  Enter a custom movie storyline to receive similar movie recommendations.
-  Search for an existing movie and discover similar movies.
-  Get recommendations using TF-IDF Vectorization and Cosine Similarity.

---

## Features

- IMDb 2024 Movie Data Scraping using Selenium
- Data Cleaning and Preprocessing
- NLP Text Processing
- TF-IDF Vectorization
- Cosine Similarity Recommendation Engine
- Storyline-Based Recommendations
- Movie Name-Based Recommendations
- Interactive Streamlit Web Application
